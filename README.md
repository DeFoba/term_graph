# Term Graph & Scientific Annotation

Проект посвящён автоматическому выделению терминов и построению графа понятий
на основе научных публикаций с использованием нейросетевых и статистических методов.

## Возможности
- Извлечение терминов из текста
- Использование авторских ключевых слов
- Построение графа терминов
- Аннотирование публикаций
- Визуализация связей

## Стек
- Python
- NLTK
- Scikit-learn
- Sentence-BERT
- NetworkX
- Matplotlib

## Структура проекта
/data        - статьи  
/src         - код  
/experiments - эксперименты  
/models      - модели  
README.md

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
